# AI Analyzer Website

## Scenario
The **AI Analyzer** is a modern, responsive web application designed to help users track their expenses, analyze spending trends, and receive personalized financial advice through an AI-powered chat interface. Built with a focus on UI/UX excellence, it features a glassmorphism aesthetic and intuitive interactions.

## Features
- **Expense Tracking**: Add, edit, and delete daily expenses with categories.
- **Visual Analysis**: Interactive charts (Pie & Line) to visualize spending habits.
- **AI Insights**: Real-time feedback on your financial health based on spending data.
- **AI Chat Advisor**: "Speak With Me" feature to ask questions and get voice-enabled responses.
- **Modern UI**: Glassmorphism design, Dark/Light mode toggle, and smooth animations.
- **Responsive**: Fully functional on mobile, tablet, and desktop.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom styles with CSS Variables, Flexbox, Grid.
- **Bootstrap 5**: Responsive layout and components.
- **JavaScript (Vanilla)**: Core logic, DOM manipulation, localStorage.
- **Chart.js**: Data visualization.
- **Google Fonts**: Typography.

## Design System

### Color Palette
- **Primary**: `#6C63FF` (Purple)
- **Secondary**: `#00BFA5` (Teal)
- **Accent**: `#FF6584` (Pink/Red)
- **Background**: Dark (`#121212`) / Light (`#F0F2F5`)

### Typography
- **Headings**: `Sora` (Weights: 400, 600, 700)
- **Body**: `Inter` (Weights: 300, 400, 500, 600)
